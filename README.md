# Orange Juice Landing Page

A scroll-driven landing page for Orange Juice — pure and natural. Built with React 18, Vite, and GSAP (ScrollTrigger) for hero animation and section transitions.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Then open the URL shown in the terminal (e.g. http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Stack

- **React 18** + **Vite 5**
- **GSAP 3** with ScrollTrigger for scroll-linked hero frame animation and phrase transitions
